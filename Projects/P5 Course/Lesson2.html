Comments

It's a good habbit to put some lines of comment in your code to explane what the perpose is of the code. So when you read your code back, it is easier to figure out how it works.

You can put code on a seperate line or on the back of a line after the ;

Comments can make code a lot easier to read!

              //this is a line of comment.
              ellipse(30,100, 10,10); //eye

fill, stroke and strokeWeight

But before we can draw a line or a rect we have to get a color. This time you need a color for the stroke. And of course you can use the fill function. 
                fill(color);
                stroke(color);
                strokeWeight(2);

In this lesson we will use only the black, white and gray colors. fill(200) will give your shapes a lightgray fillcolor.
stroke(0) will give your shapes a black outline.
stroke(255) will give your shapes a white outline.
You can change the thickness of your line with the function strokeWeight.
strokeWeight(10), will give your shape a big outline.

Pay attention to the capital in the function name. All functions in p5 start with a small letter, and the words are written together, but every next word will start with a capital.

If you don't want to use a fillcolor are a stroke. You can call the functions:

                noFill();
                noStroke();

Ellipses and cirles

In the first lesson we learn to use a function to draw ellipses and circles. 

              ellipse(x, y, width, height);

Rectangles and squares

So the next function will not be so hard to understand.

              rect(x, y , width, height);

With the rect function you can draw rectangles on a position on the canvas.
The rect function needs to know where his top-left corner is.
The width and height params control the size of the rectangle.
If the width is the same as the height value, you can draw a square.

Triangles

The triangle function needs the 3 positions (x,y) of his corner.
And the quad (quadrilateral) function needs 4 positions (x,y) of his corner

              triangle(x1, y1 , x2, y2, x3, y3);

Like an ellipse and a rectangle, the triangle has an innershape and an outline.
Sou you can use the fill function to fill the triangle and the stroke function and strokeWeight function for the outline.
