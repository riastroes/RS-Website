Example 2

    function setup() {
      createCanvas(400,300);
    }

    function draw() {

      stroke(random(255));
      strokeWeight(10);
      point(random(400), random(300));
    }

the draw function is called every
second 60 times.
Draw a point at position(x,y)
x = a random value between 0 and 400
y = a random value between 0 and 300

give the stroke a random color
between 0 (black) and 255 (white)
