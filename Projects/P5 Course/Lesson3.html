Basic drawing functions

Background

              //200 is light gray
              background(200);

You can set the background in the setup function. Now it is executed once.
If you put it in de draw function, it will be executed every time the draw function is called by p5. So all that was drawn will be overwritten.

Point and lines

              point(x, y);
              line(x, y, x1, y1);

You can set the size of the point with the strokeWeight function.
If you don't set the strokeWeight, it will be the size of 1.
We call this a default value.
The default value voor stroke is black and the default value for fill is white.

The line function will draw a line from position (x, y,) to position (x1, y1); You have to fill in numbers for the x and y, the x1 and the y1.

If you copy this example to your p5 editor, you will notice round line ends.
You can control this with a function strokeCap. Go to the reference of p5 and find out how.

Variables

If you want to change a line just a little bit. You need a variable. Think of a variable as a box where you can store a value.

You already know a two variables of p5

  • mouseX
  • mouseY
They store the x and y value of mousepointer.
  • width
  • height
store the value of the width and height of your canvas.
  • windowWidth
  • windowHeight
store the width and the height of the window of the user.
So now you can create a canvas as big as your window, even if you don't know the width of your screen. 

            function setup(){
              createCanvas(windowWidth, windowHeight);
            }

If you create your own variables in a function. The variable will only be known within this function. If you declare your variables outside the functions. You can use the variable in all your functions. 

              var x;
              function setup(){
                 createCanvas(400,300);
                 x = 0;
              }
              function draw(){
                x = x+5
                point(x,y);
             }

In this example the x is declared outside the draw function on top, because in this example we will use te x in the draw function, but before you can add a value to a variable. This variable has to have a value. This principle we call initialization. The setup function is the right place to do some initializations. In the draw function every time this function loops we will add 5 to the value of x. You also can write it like this 

              x += 5;
              x -= 5;

Animation and frames

Every time the draw function is called, a frame is drawn to the canvas. The maximum frameRate is 60 times a second. Sometimes this is to fast and you want to slow down your animation. You can set the maximum frameRate a bit lower in the setup function

          function setup(){
             createCanvas(400,300);
             frameRate(10);
          }

If you want to animate a shape you will need variables.

Lets say you like a rectangle to move from left to right

You can find this example on the example page of this lesson.

            var x;
            function setup(){
               createCanvas(400,300);
               x = 0;
            }
            function draw(){
              rect(x,100, 100,40);
              x += 5;

           }

Here the rectangle has a speed of 5 pixels a frame.
