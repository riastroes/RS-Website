What can you learn?

Before we can start I have to tell you some things, so you understand the tools, we will use to be creative on the web.

What is programming?

Programming is sending instructions to a computer. To do this we have to learn a language the computer understands.

Javascript is a collection of functions and values the computer can read and execute. All the commands are easy to read in English, but inside the computer they are translated to other languages, the computer can understand.

Javascript is a programming language that is used on the computer for displaying data in the browser on the web.

We will learn a little bit of javascript, so we can start sending instructions to the computer

What do you need to know about functions?

Functions are a set of instructions. Together these instruction execute a specific task in your computer. Every instruction can be a value or an other function. In a function all the instructions are executed after each other every time in the same order.

A function has a name and in parentheses some params and between two braces a set of instructions.

function name (param1, param2, etc.){
...
a set of instructions
...
}

Not every function has params and later on we will see how we can use these params.

What are params?

Params are values, you can give to a function to work with.

p5 starts with two functions setup and draw, both functions don't have any params. These functions need no values to work with, they just execute the instructions within the braces.

What are values?

A value can be a number, a char, word, string, an object or the result of a function. We will see examples of all these different values in this course. The most important values in a computer are 0 and 1. The computer can calculate with these to numbers, but 0 and 1 also can represent the values false and true. We don't need to know how the computer is calculating with these numbers. Other smart programmers created functions for us to make it easy to do far more interesting things with our computers. Javascript is a collections of these functions and p5 is a combination of instructions in javascript that we can use to write instructions to the computer and tell the computer what and how to draw on a canvas on a webpage in the browser.

Introduction to p5

p5.js is een javascript library, developed special for artist graphic designers, education and beginners. It is a collection of functions that allow you to draw on a canvas element in a webpage in your browser.

What can you create?

Getting started ...

The p5 editor is starting with two functies, the function setup and the function draw.

function setup

In this function you can make all the preparations necessary for your sketch, soon we will see what kind of preparations you can make for a sketch. Remember that this function is executed only once. This means that al the functions between the accolades will be executed one time,after each other.

function draw

In the function draw the actual drawing will take place. This function is executed 60 times a second. This means all the functions between the accolades of the draw function will be executed after each other 60 times in a second.

Example

          function setup(){
            createCanvas(800,600);
          }

In this function setup we create a canvas element in the webpage. This function needs two params to control the width and the height of the canvas. The canvas will be 800 pixels width and 600 pixels height. You can address every position on the canvas with an x, y value. 

            function draw(){
              fill(random(255));
              ellipse(mouseX, mouseY, 100,100);
            }

In de function draw we will tell the computer to draw an ellipse.
Before we can draw we need to get a color to work with. We want to fill the ellipse with a random color, with a value between 0 and 255. 0 represents black and 255 represents white. A random color between black and white is a color gray.

The function ellipse will draw a circle or ellipse on the canvas. This function needs four params;
the first two params are the position on the canvas (x,y).
Here we use the position of the mouse, to get a x and y value.
The position of the mouse are stored in mouseX and mouseY.
The third and the last params are the width and the height of the ellipse.
