What can you learn?

There are a few different colorsystem in P5. We will learn about the RGB colormode.
RGB, stands for Red, Green and Blue. The three primary colors of an additive colorsystem
In an additive colorsystem you mix the colors with light and the three colors Red, Green and Blue will give you a white light.

color(r,g,b)

                 var c = color(r,g,b);

Here we say, the return value of the function color, with the params r,g,b returns a color value, we store this value in the variable c. The params r, g, b have a value between 0 and 255. So the minimum amount of light (black) is color(0,0,0) And white is represented by color(255,255,255); We don't allways have to use the color function. In de functions background and fill and stroke, you can use the r, g, b params without the color function. 

                 fill( r, g, b);
                 stroke (r, g, b);
                 background (r, g, b);

                 //but you also can write
                 var c = color(r,g,b);
                 fill(c);

Transparency

If you want to use transparent colors for your shapes, then you add an extra parameter to the color function with the alpha value between 0 and 255

              var c = color(r,g,b,a);
              fill(r,g,b,a);

What can you create?

Color mixing

On the internet you can find a lot of resources for nice color combinations, but I advice you to start color mixing your self. Buy some cheap paint and start mixing. Look for the combinations you like! Don't use black. It kills all the colors, instead mix a dark color with blue and brown.

Look at pictures in magazines and see if you can find color combination you like and use them in your next P5 sketch.
